# CustomEventsDG

CustomEventsDG is a source-available Minecraft plugin for custom random server events.

The plugin creates a temporary rift event at random world coordinates. Players can find the event zone, fight mob waves, activate crystals, defeat a mini-boss and receive rewards from a chest.

## Features

- Random event spawn coordinates
- Temporary rift event zone
- Mob waves
- Three activation crystals
- Mini-boss with bossbar
- Reward chest
- Particle and sound effects
- Admin commands
- Configurable settings and loot
- Paper/Spigot 1.16.5
- Java 16
- Maven project

## Commands

```text
/cevent start
/cevent start <world> <x> <z>
/cevent stop
/cevent tp
/cevent info
/cevent reload
```

## Permissions

```text
customevents.admin
customevents.play
```
